Views Sandbox

This is a Django 2 web application which serves as a sandbox, test area, and reference by example for working with Django views.

Class-based Views

Simple - is an example of a very basic view returning html text using the HttpResponse method.

cbvlist - is an example of a basic Class-based View(CBV) that is a list view, that includes a Django-based search on first or last names.

cbv sortable table list - same list view as above but using Django-tables2 which has sortable columns.

cbv custom sortable table list - same list view as above but using Django-tables2 with a Table class that allows picking which sortable columns to show.

cbv custom filtered / sortable table list - same list view as above using Django-tables2 with a Table class with sortable columns AND filtered by status.

Function-based Views

fbvlist - is an example of a basic Function-based View(FBV) that is a list view, that includes a Django-based search on first or last names.

Form Views

form_test - is a form using the Form class (forms.Form).

model_form_test - is a form using the ModelForm class (forms.ModelForm).
